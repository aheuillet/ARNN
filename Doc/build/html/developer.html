Developer Guide¶

In regards to the code used to generate and run echo state networks, refer to the Github in the Useful links. It contains several files as the code genereting the tank of networks (ESN.py), a example file (simple_example_MackeyGlass.py) and the code used to compute the metrics which can be displayed (observables.py). Futhermore, a README.md file is found explaining the functionalities as documentation.
