In order to install all the depencies for ARNN developers, the shell script */Code/Django/recurView/setup.sh* can be executed. This script is designed to work on Debian-based systems (such as Ubuntu).

Then several python modules can be required to be installed such as SKlearn, Numpy, Scipy or Bokeh. If a module is not installed by the bash script *setup.sh*, it can be done using the command **pip3 install module_name**.
